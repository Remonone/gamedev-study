# Smart Object Picker for Unity

Quickly select overlapping objects in the Scene view with a modifier key + click.

## Features

- **Modifier + Click selection** - Hold Alt (or custom modifier) and click to show all objects at that position
- **Visual popup menu** - See all overlapping objects in a scrollable list with icons
- **Object highlighting** - Hover over items to highlight them in the scene with a bounding box
- **Customizable highlight color** - Choose your own wireframe highlight color
- **Adaptive wireframe grid** - Large objects display a subdivided grid for better visibility
- **Keyboard navigation** - Use arrow keys to navigate, Enter to select, Escape to close
- **Select All** - Click the header to select all objects at once
- **Configurable settings** - Choose modifier key, highlight color, excluded layers, and selection behavior
- **Focus on select** - Optionally frame the selected object in the Scene view
- **2D & UI support** - Works with 2D colliders, sprites, and UI elements (RectTransforms)

## Installation

1. Import the package into your Unity project
2. The tool activates automatically - no setup required

## Quick Start

1. Hold **Alt** and **click** anywhere in the Scene view
2. A popup appears showing all objects at that position
3. **Hover** over items to highlight them in the scene
4. **Click** an item to select it, or use **arrow keys** + **Enter**
5. Press **Escape** or click outside to close

## Settings

Access via **Tools > Clever Clicker > Settings**

- **Modifier Key** - Alt, Control, Shift, or Command
- **Selection Behavior** - Select only, or select and focus
- **Highlight Color** - Customize the wireframe highlight color
- **Excluded Layers** - Ignore objects on specific layers

## Keyboard Shortcuts

- **↑/↓ Arrow Keys** - Navigate the list
- **Enter** - Select highlighted object
- **Escape** - Close popup
- **Click header** - Select all objects

## Requirements

- Unity 2021.3 LTS or later
- Works with all render pipelines (Built-in, URP, HDRP)
